# KakaoTalk Ad Remover

KakaoTalk Ad Remover는 카카오톡 PC 클라이언트 하단 광고 영역을 제거하는 툴입니다. 

## 사용 방법

배포된 실행 파일을 실행하세요.
툴은 백그라운드에서 동작하며 클라이언트의 광고를 비활성화하고 광고가 차지하던 영역을 제거합니다. 

윈도우 시작 시 자동 실행 기능을 지원하며, 시스템 트레이에서 우클릭하여 설정할 수 있습니다.

툴을 종료하려면 시스템 트레이에서 우클릭하여 종료하세요.

## 안내사항

배포된 툴은 Windows x64를 지원합니다.
